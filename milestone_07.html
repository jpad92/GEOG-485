BBOX height = (maxy) - (miny) = (36.5) - (34.5) = 2 degrees
BBOX aspect ratio = (BBOX width) / (BBOX height) = (3) / (2) = 1.5
Therefore, the aspect ration of this geographic region is 1.5.
